# 📦 Sistema de Gestión de Productos

Proyecto desarrollado en Python para registrar y visualizar productos desde la consola.

</div>
<h2>
  <img src="https://user-images.githubusercontent.com/74038190/216649417-9acc58df-9186-4132-ad43-819a57babb67.gif" width="50">  Descripción
</h2>  

Este proyecto es un programa desarrollado en **Python** que permite registrar y visualizar productos desde la consola.
El sistema almacena información básica de los productos, como **nombre, precio y cantidad**, utilizando estructuras de datos simples como listas y diccionarios.

El objetivo del programa es practicar conceptos básicos de programación como el uso de funciones, manejo de datos y validación de entradas del usuario.

---

# ⚙️ Funciones del programa

### 🔢 `entero()`

Esta función se encarga de solicitar al usuario un número entero desde la consola.
Incluye validación para evitar errores cuando el usuario ingresa datos que no son numéricos.

---

### 🔢 `decimal()`

Permite ingresar números decimales (float).
Se utiliza principalmente para registrar valores como el **precio de los productos**, asegurando que el dato ingresado sea válido.

---

### ➕ `agregar_producto()`

Esta función permite registrar nuevos productos dentro del sistema.
Solicita al usuario la información del producto y posteriormente la almacena en una lista para poder gestionarla dentro del programa.

---

### 📋 `listar_Productos()`

Muestra en pantalla todos los productos que han sido registrados.
La información se presenta en forma de **tabla**, lo que facilita visualizar los datos almacenados.
